# Portal Message Format

Use this exact shape for every drafted message:

```
Subject: <topic> - <urgency: routine | this week | urgent>

Hello Dr. <last name>,

<one-sentence ask>

<2-4 sentences of relevant clinical context: onset, duration, severity,
triggers, what's been tried, change since last visit>

<bullet list of pertinent data, only if relevant>

Thank you,
<patient first name>
```

## Rules

- **Subject line**: topic + urgency tag. Urgency = `routine` | `this week` | `urgent`.
- **Greeting**: `Hello Dr. <last name>,` if known, else `Hello [Care Team],`.
- **First body sentence**: the ask. No throat-clearing.
- **Context paragraph**: only what the clinician needs to act on the ask.
- **Bullets**: only if there are 3+ data points (meds, vitals, dates) worth
  listing distinctly.
- **Sign-off**: `Thank you,` then patient first name only.
- **Word budget**: aim for ≤150 words; hard cap 220.
- **Tone**: respectful, direct, calm. No apologies for "bothering" the team.

## Pertinent data to include when relevant

- Current medication name + dose + start date
- Home vitals (BP, HR, temperature, weight) with date
- Recent lab/imaging result the patient is asking about (with date)
- What the patient has already tried (OTC meds, rest, dietary changes)
- Date of last related visit or test
